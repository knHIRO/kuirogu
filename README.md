# kuirogu

## サイト概要
### サイトテーマ
地元ではない土地勘のない場所でおすすめの食べる所の情報を検索したり、自分が行った店の投稿したりできるSNSサイトです。

### テーマを選んだ理由
旅行と食べることが趣味で新しい場所でいろんな店に行くことがが多いです。しかし自分に合わない店に遭遇することがしばしばあり、他の人がどのような場所でおすすめの店が知れて参考になるサービスがあれば便利だと考え、このテーマにしました。

### ターゲットユーザ
旅行先の食べるところを探している人

### 主な利用シーン
その場所のおすすめの店を検索する時、自分が行った店の感想、おすすめ等を投稿する時
